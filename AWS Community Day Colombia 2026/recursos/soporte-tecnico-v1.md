# Manual Interno de Soporte Técnico

## Problemas de conexión

Si un usuario no tiene internet:

1. Verificar Wi-Fi o cable.
2. Reiniciar conexión.
3. Verificar si otros usuarios tienen el problema.
4. Abrir ticket.

## Computador lento

1. Guardar trabajo.
2. Reiniciar el equipo.
3. Cerrar aplicaciones innecesarias.
4. Contactar soporte.

## Acceso SAP

El acceso a SAP se solicita así:

1. Abrir ticket interno en Jira. En el ticket debes indicar área, rol, correo de @cloudopsguild.com
2. Seleccionar “Solicitud SAP”.
3. Esperar aprobación del jefe.
4. Esperar aprobación de Seguridad.
5. El equipo de TI se contactará contigo cuando esten las aprobaciones

## Restablecimiento de contraseña

Usar el portal interno.

## Regla de soporte

Responde únicamente
con información de este documento.
Si no existe información,
indica que no sabes.
